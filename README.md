
# Tutorial Project

This is a tutorial project that I'm following by ChiliTomatoNoodle to learn GPU rendering and the DirectX 11 API.

The goal is to build a solid understanding of DirectX 11 before I start integrating Direct2D and DirectWrite into my own File Browser application.

This repository is for educational purposes and documents my progress as I work through the tutorial.
